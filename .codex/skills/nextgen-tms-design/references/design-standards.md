# Design Standards

## Visual Tone
- Modern logistics control panel
- Neutral base palette
- Status colors used intentionally, not everywhere
- Clear typography hierarchy over visual effects

## Page Composition
- Start with a page header and one sentence of operational context.
- Group content by workflow stage.
- Use cards when they create separation; avoid card spam.

## Tables
- Optimize for dispatch scanning.
- Order columns by operational importance, not schema order.
- Show identifiers, origin/destination, assignee, and status early.

## Forms
- Labels are required.
- Submission state is required.
- Validation and error feedback must be visible and specific.
- Do not depend on placeholder text as the only guidance.

## Empty States
- Explain what is missing.
- Tell the user what action to take next.
- Keep the message short and operational.

## Status Presentation
- Shipment status should be visually consistent across list and detail pages.
- Exceptions and cancellations should be unmistakable.
- Avoid using too many competing badge styles.
