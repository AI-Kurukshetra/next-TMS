You are a senior frontend engineer and product designer.

Build a beautiful, modern landing page for a logistics web application called:

"NextGen Transportation Management System (TMS)"

Tech stack:

- Next.js 15 (App Router)
- React
- Tailwind CSS
- Typescript

Goal:
Create an impressive landing page suitable for a hackathon demo.

Design requirements:

The landing page should include the following sections:

1. Hero Section

- Large headline introducing the platform
- Subheading explaining logistics optimization
- Call to action buttons:
  - "Get Started"
  - "View Dashboard"
- Logistics themed background (map routes / delivery illustration)

2. Features Section
   Show cards for main features:

- Shipment Management
- Real-time Tracking
- Carrier Management
- Driver & Vehicle Management
- Route Optimization
- Analytics Dashboard

Each card should have:

- icon
- title
- short description
- hover animation

3. How It Works Section
   Show workflow steps:

Customer → Shipment → Carrier → Driver → Delivery

Use visual timeline or step cards.

4. Dashboard Preview Section
   Show a mock dashboard UI screenshot or component.

Include metrics like:

- Total Shipments
- Active Drivers
- In Transit Deliveries
- On-time Delivery Rate

5. Benefits Section

Show benefits such as:

- Reduce logistics cost
- Real-time shipment visibility
- Optimized route planning
- Carrier performance analytics

6. CTA Section
   Encourage users to start managing shipments.

7. Footer
   Include:

- product links
- company info
- GitHub link
- contact

Design Style:

- clean modern SaaS design
- large typography
- gradient backgrounds
- smooth hover effects
- responsive mobile design

Technical Requirements:

- Use Next.js App Router
- Create page at: /app/page.tsx
- Create reusable components in /components
- Use Tailwind utility classes
- Use icons from lucide-react
- Add subtle animations and hover effects

Output:
Generate complete code for:

- landing page
- reusable feature card component
- navbar component
- footer component

Focus on clean UI and good UX.
